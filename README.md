# Terraria

## Install Server
```bash

sudo apt update && sudo apt install -y wget tmux unzip
wget https://terraria.fandom.com/wiki/Server#Downloads
unzip terraria-server*
cd 1412
cd Linux
chmod +x TerrariaServer.bin.x86*
./TerrariaServer.bin.x86_64

``` 

### Running Server
./TerrariaServer.bin.x86_64
